# BACKLOG — 待办唯一账本 / Single source of open items

> **账本与日志分离 (ledger-journal split)**:`PROGRESS.md` 是叙事日志(可以长);
> 本文件是待办账本(**必须短**,保持 <100 行,任何新 session 一口气读完、无从截断)。
> 铁律 / Iron rules:
> 1. PROGRESS 里每写「下一步 / 待 / 暂缓 / 承诺」,**必须同步登记一行到 OPEN**;
> 2. 关闭时移入 CLOSED 并**回写凭据**(commit 短哈希 / 日志条目号),**只增不删**
>    (CLOSED 过长时整段归档进 PROGRESS 一条,此处留指针);
> 3. 宣布「无缺口 / 全部完成」的**前置条件 = OPEN 为空,或逐条说明**。
